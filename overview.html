🔄 Workflow Hệ Thống

1. Load Bankroll & Config
2. Thu thập dữ liệu mới nhất (FIFA, Form, Injuries, Travel...)
3. Feature Engineering
4. Chạy Model → Dự đoán xác suất
5. Crawl Polymarket → Lấy Implied Probability
6. Tính Edge (≥ 7%)
7. Nếu có edge → Tính Position Size bằng 1/4 Kelly
8. Ghi log + Gửi Telegram Alert
9. Lưu lịch sử suggestion
